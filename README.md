# Nirmalyam Krafts - Frontend Server

This repository contains the frontend server code for the Nirmalyam Krafts application.

## 🚀 Getting Started

### Prerequisites

- Node.js (v18 or higher)
- npm

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/Nirmalyam-Krafts-Frontend.git
   cd Nirmalyam-Krafts-Frontend
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

### Usage

Start the development server:

```bash
npm run dev
```

The application will be available at `http://localhost:5173`.
